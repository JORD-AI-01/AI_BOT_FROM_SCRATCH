# AI_BOT_FROM_SCRATCH


A fully offline AI chatbot built using open-source large language models.
No API keys. No cloud services. Full local inference and fine-tuning.

## Goals
- Run an LLM completely offline
- Implement chat memory
- Fine-tune the model using LoRA
- Learn how modern AI training works internally

## Tech Stack
- Python
- PyTorch
- HuggingFace Transformers
- PEFT (LoRA)
- CUDA (NVIDIA GPU)

## Hardware Used
- GPU: NVDIA RTX 4060 8GB
- RAM: 16GB (+ swap)
- OS: Linux(NIX OS)
